# 超级贪吃蛇游戏

一个功能丰富的贪吃蛇游戏，包含多种游戏模式和道具系统。

## 游戏特色

### 🎮 四种游戏模式
1. **经典模式** - 传统贪吃蛇玩法
2. **障碍物模式** - 增加障碍物，增加挑战
3. **传送门模式** - 使用传送门瞬间移动
4. **生存模式** - 包含障碍物、传送门和生命系统

### 🍎 多种食物类型
- **普通食物** (🍎) - +10分
- **钻石食物** (💎) - +50分  
- **毒物** (☠️) - -20分

### ⚡ 道具系统
- **闪电道具** (⚡) - 减速5秒
- **生命道具** (❤️) - 恢复生命值
- **炸弹道具** (🎯) - 清除蛇尾3节

### 🏆 游戏机制
- **等级系统** - 每100分升级
- **难度递增** - 随等级提升速度加快
- **高分记录** - 自动保存最高分
- **生命系统** - 生存模式中有3条生命

## 操作说明

### 🎯 基本控制
- **方向键** ↑ ↓ ← → - 控制蛇的移动
- **空格键** - 暂停/继续游戏

### 🎮 游戏控制
- **开始游戏** - 开始新游戏
- **暂停** - 暂停/继续游戏
- **使用道具** - 使用收集到的道具（需要点击）

## 游戏规则

### 基础规则
1. 控制蛇移动吃食物
2. 吃到食物蛇会变长，得分增加
3. 撞到墙壁、自己或障碍物游戏结束

### 不同模式规则

#### 经典模式
- 基础玩法
- 随着分数增加，游戏速度会加快

#### 障碍物模式  
- 地图中会随机生成障碍物
- 撞到障碍物游戏结束
- 随等级提升障碍物增多

#### 传送门模式
- 地图中有紫色和蓝色传送门
- 走进紫色传送门会从蓝色传送门出来
- 传送门会随等级增加

#### 生存模式
- 包含障碍物和传送门
- 有3条生命
- 撞到障碍物会损失一条生命
- 吃到生命道具可以恢复生命

### 道具说明
- 闪电道具：让游戏速度减慢5秒
- 生命道具：增加一条生命（生存模式）
- 炸弹道具：清除蛇尾3节
